Welcome to OGLplus’s documentation!¶

OGLplus is a collection of utilities useful, when writing modern, portable C++ applications which do GPU rendering and audio playback and capture.

Note

This documentation is work-in-progress and incomplete.

For the API reference see the Doxygen-generated docs.
